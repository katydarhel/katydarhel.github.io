We are constantly adding new features based on the wishes of our customers. All the added functionality is immediately available to other customers of our service without additional surcharges. Listed below are just some features of the service RoutePlanner.

AUTOMATIC SCHEDULING OF THE MOVEMENT OF GOODS AND FREIGHT

Automatic scheduling the movement of goods and cargo enables distribution/express/service companies dramatically reduce the time of planning.

Using the service RoutePlanner it is possible to automatically generate routes for all your fleet of transport unit at the same time, and will select the optimal number of machines, subject to a set of constraints, which are quite difficult to control man.

OPTIMIZATION OF TRAFFIC ROUTES

The result of the algorithm of automatic planning are optimized routes for each unit of transport, with an indication of the order of visits couteragents in the same flight and indicating the planned arrival time.

CONTROL THE VOLUME-WEIGHTED PERFORMANCE OF VEHICLES

When calculating the optimal routes, necessarily take into account the dimensions of the load (weight, volume) and is monitored load of the body.

AUTOMATIC GEOCODING ADDRESSES COUNTERAGENTS

All available counteragents addresses can be loaded into the system automatically. RoutePlanner recognizes most addresses. If the address is set incorrectly, or it is not yet in the map (new streets, new shops and so on), the system marks them and prompt the user to perform manual adjustment. Manual adjustment mode has a very simple and friendly user-interfacem the system constantly helps the user in this mode, so that manual editing will not take long.

TRANSPORT MONITORING MODULE

Transport monitoring module works on smartphones drivers in delivery and allows you to receive the following operation information:

  • The current location of each car
  • GPS coordinates, the actual route
  • The total current pass rate for all cruises of the day

Tracing module execution orders for delivery allows both professionals and Logistics, and representatives of the contractors to monitor the degree of execution of orders for delivery via the Internet. The system allows you to track the current status of individual flights, reguests for delivery.
